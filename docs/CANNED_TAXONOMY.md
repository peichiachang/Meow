# 300 則罐頭句型分類說明

依指定分類法對 300 則罐頭進行關鍵字掃描，統計各分類詞彙的**出現次數**（一則可同時命中多個詞）。

## 分類架構

```json
{
  "subject": ["本喵", "我", "本大爺", "主子"],
  "address": ["奴才", "鏟屎官", "這隻人類", "餵飯的"],
  "state": ["肚子餓扁了", "電力剩 1%", "看透貓生", "心情極好", "覺得你很煩"],
  "verb": ["開罐頭", "抓沙發", "踩肚子", "哈氣", "蹭腿", "理毛"],
  "ending": ["懂了嗎？", "不准賴床！", "真拿你沒辦法", "喵嗚！"],
  "action": ["(甩尾巴)", "(發出呼嚕聲)", "(瞇眼睛)", "(舔爪子)"]
}
```

## 統計結果摘要

| 分類 | 詞彙 | 出現次數 |
|------|------|----------|
| **subject** | 本喵 | 235 |
| | 我 | 159 |
| | 本大爺、主子 | 0 |
| **address** | 奴才 | 161 |
| | 鏟屎官 | 111 |
| | 這隻人類、餵飯的 | 0 |
| **state** | 覺得你很煩 | 1 |
| | 其餘 | 0 |
| **verb** | 哈氣 | 15 |
| | 開罐頭 | 6 |
| | 抓沙發、理毛 | 2 |
| | 踩肚子、蹭腿 | 0 |
| **ending** | 真拿你沒辦法 | 1 |
| | 其餘 | 0 |
| **action** | (瞇眼睛) | 17 |
| | (甩尾巴) | 16 |
| | (舔爪子) | 9 |
| | (發出呼嚕聲) | 6 |

## 說明

- **subject / address**：300 則裡以「本喵」「我」「奴才」「鏟屎官」為主；「本大爺」「主子」「這隻人類」「餵飯的」目前未出現。
- **state / ending**：多數為你列出的「精準句」；句型中較常出現的是同義或相近說法（例如餓、累、煩的其它寫法）。
- **verb**：罐頭裡較多「蹭你的腿」「用頭撞你」等，與「蹭腿」「踩肚子」用字不盡相同，故部分 verb 次數較低。
- **action**：括號內動作（甩尾巴、瞇眼睛等）已支援全形／半形括號比對。

## 如何重新產生

```bash
npm run classify:canned
```

會更新 `docs/canned-taxonomy-classification.json`（含每則罐頭的前 80 字與標註到的分類）。

若要把「蹭你的腿」「踩你的肚子」等也納入統計，可在 `scripts/classify-canned-by-taxonomy.ts` 的 `TAXONOMY.verb` 中加上這些同義詞再跑一次。
